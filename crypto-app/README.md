## Installation and Setup Instructions


Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install 
react-router-dom
antd @ant-design/icons 
react-redux @reduxjs/toolkit 
axios 
chart.js 
html-react-parser 
millify 
moment 
react-chartjs-2`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000/`
